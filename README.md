Official core plugin for TitanPrison. 

##TitanPrison 
is/was a free-to-play and fair for all overpowered prison server. TitanPrison is/was intended to push the limits of what can reasonably be achieved through extremely well-paced progression and a ceiling higher than ever.


##TitanCore
is the core plugin which contains all custom functionality TitanPrison used.

Maven build script is included so anyone can compile the program. The compiled version is available [here](https://github.com/MadCuzDev/TitanCore/blob/master/TitanCore/target/TitanCore-1.0.jar) but is not guaranteed to be updated.

**Features:**
- Extremely powerful enchantments with unique and performant functionality
- Currencies and custom economy
- Integreation into LuckPerms and WorldGuard API
- Quality of life features such as no hunger
- Rank system and integrated progression
- Multiple reset layers (rank, prestige, and mastery)
- Visual customization for chatting based around rank, prestige, and mastery
